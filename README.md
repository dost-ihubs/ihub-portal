# iHub Portal

The **iHub Portal** is a web-based directory and interactive map for the Department of Science and Technology (DOST) Innovation Hubs (iHubs) in the Philippines. It allows users to explore launched Regional and Provincial Innovation Hubs, view their locations, and access essential information about each hub.

## Features

* 🗺️ Interactive map of DOST Innovation Hubs across the Philippines
* 📍 Search and filter iHubs by region, province, or hub name
* 🏢 View detailed iHub profiles, including:
  * Location and address
  * Contact information
  * Regional and Provincial Focal Persons
* 🔗 Access official links and social media pages
* 📚 Browse resources and other relevant information about the iHubs Program
